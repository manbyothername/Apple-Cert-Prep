# APT Apple Device Support Practice Exam (Dynamic)

## Run locally
Open the folder in VS Code, then use **Live Server** extension:
- Right click `index.html` → "Open with Live Server"

Or use any static server.

## Deploy to GitHub Pages
1. Push this folder to a GitHub repo.
2. Repo Settings → Pages
3. Source: `main` branch / root
4. Visit the Pages URL.

## Notes
- Questions are original practice prompts (not copied from any official test).
- The app stores your performance in localStorage and adjusts topic weighting.
